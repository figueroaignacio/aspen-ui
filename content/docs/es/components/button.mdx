## Cómo usar

Para usar el componente `Button`, impórtalo y agrégalo en tu JSX como se muestra a continuación:

```jsx
import Button from "@/components/ui/Button";

const MiComponente = () => {
  const manejarClick = () => {
    alert("¡Botón clicado!");
  };

  return (
    <Button onClick={manejarClick} className="bg-blue-500 text-white rounded">
      Haz Clic
    </Button>
  );
};

export default MiComponente;
```
