## Cómo usar

Para usar el componente `Card`, impórtalo y agrégalo en tu JSX como se muestra a continuación:

```jsx
import Card from "@/components/ui/Card";

const MiComponente = () => {
  return (
    <Card className="shadow-lg p-4 rounded-lg bg-white">
      <h2 className="text-xl font-semibold">Título de la Tarjeta</h2>
      <p className="text-gray-600">
        Esta es una descripción dentro de la tarjeta.
      </p>
      <Button className="mt-4">Aprender Más</Button>
    </Card>
  );
};

export default MiComponente;
```
