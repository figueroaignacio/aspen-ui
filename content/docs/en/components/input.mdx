## Usage

To use the `Input` component, import it and include it in your JSX as shown below:

```jsx
import Input from "@/components/ui/Input";

const MyForm = () => {
  return (
    <form>
      <Input
        type="text"
        placeholder="Enter your name"
        className="border rounded-lg p-2"
      />
    </form>
  );
};

export default MyForm;
```
