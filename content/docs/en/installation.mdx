## Requirements

To use Aspen UI, ensure you have the following in your project:

- **React**: A JavaScript library for building user interfaces.
- **Tailwind CSS**: A utility-first CSS framework for creating custom designs.
- **clsx**: A utility for constructing `className` strings conditionally (optional).
- **tailwind-merge**: A utility for merging Tailwind CSS classes (optional).

## How to Use Aspen UI

Aspen UI is a kit of user interface components that you can easily integrate into your project by copying and pasting the provided code snippets.

### Steps to Get Started

1. **Copy the Code**

   - Navigate through the Aspen UI documentation to find the component you want to use.
   - Each component will have its code snippet available for easy copying.

2. **Paste into Your Project**

   - Insert the copied code directly into your React component file.
   - Ensure that you include any necessary imports for the component styles.

3. **Customize as Needed**
   - Modify styles and properties according to your project's design requirements.
   - Use Tailwind CSS utilities to further customize the look and feel of the components.
