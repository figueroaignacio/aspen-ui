## ¿Instalación?

Como se mencionó en la introducción, I7A/UI no es una librería en el sentido tradicional, sino una colección de componentes reutilizables que puedes incorporar directamente en tu proyecto. Por ello, no existe una instalación única para todo el conjunto. En su lugar, cada componente incluye su código fuente disponible en la documentación, listo para ser copiado, adaptado y utilizado según tus necesidades.

<Callout variant="tip">
  <CalloutTitle>Aunque...</CalloutTitle>
  <CalloutDescription>
    Para que los componentes funcionen correctamente, es necesario instalar
    algunas utilidades complementarias que trabajan junto con CSS. A
    continuación, se detallan las que debes tener en cuenta.
  </CalloutDescription>
</Callout>

## Utilidades

<Steps>

  <Step>
    Instala las siguientes utilidades para que los componentes funcionen correctamente:

    ```bash
    pnpm i clsx tailwind-merge
    ```

  </Step>

  <Step>
    Crea un archivo de utilidades en tu proyecto, por ejemplo, `utils.ts`, y copia el siguiente código:

    ```ts
    import { clsx, type ClassValue } from "clsx";
    import { twMerge } from "tailwind-merge";

    export function cn(...inputs: ClassValue[]) {
      return twMerge(clsx(inputs));
    }
    ```

  </Step>

</Steps>
