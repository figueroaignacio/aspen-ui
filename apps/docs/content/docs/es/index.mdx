## ¿Que es I7A/UI?

Inspirado en herramientas como [Tailwind](https://tailwindcss.com), [Shadcn](https://ui.shadcn.com), [Radix](https://radix-ui.com) y [Chakra UI](https://chakra-ui.com), I7A/UI es un conjunto de componentes de interfaz de usuario que se pueden utilizar para crear aplicaciones web modernas y atractivas. Está diseñado para ser fácil de usar y personalizar, lo que permite a los desarrolladores centrarse en la creación de experiencias de usuario excepcionales.

## Motivación

Este proyecto nace de la curiosidad y la necesidad de construir una colección de componentes utiles tanto para mí como para otros desarrolladores que prefieren no partir desde cero al crear componentes. Su objetivo es ofrecer una base sólida para el desarrollo de aplicaciones web modernas, sin sacrificar flexibilidad ni personalización. Combinando buenas prácticas de diseño y desarrollo, I7A/UI busca simplificar la creación de interfaces de usuario atractivas, funcionales y adaptables.

## Preguntas Frecuentes

### ¿I7A/UI es una librería de copiar y pegar?

Exactamente. Este proyecto tiene como objetivo proporcionar componentes reutilizables y personalizables que puedes copiar y pegar fácilmente. Sin embargo, es importante aclarar que **NO** es una "librería" en el sentido tradicional, ya que no requiere la instalación de una dependencia.

### ¿Los componentes están diseñados exclusivamente para proyectos en React?

Aunque los componentes están construidos con React, puedes adaptarlos a cualquier framework o librería de tu preferencia. La idea principal es que el código sea reutilizable y flexible, permitiéndote ajustarlo a las necesidades de tu proyecto.

### ¿Debo usar Tailwind o CSS Modules?

La elección es completamente tuya. Los componentes son compatibles tanto con Tailwind como con CSS Modules, por lo que puedes optar por la opción que mejor se adapte a tu flujo de trabajo o preferencias.

### ¿Por qué usar TypeScript?

Recomiendo utilizar TypeScript, ya que actualmente es una práctica estándar para desarrollar proyectos más seguros y robustos. TypeScript te ayuda a detectar errores en tiempo de desarrollo y mejora la experiencia al trabajar con aplicaciones web modernas.
