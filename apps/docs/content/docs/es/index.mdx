## ¿Que es I7A/UI?

Inspirado en herramientas como [Tailwind](https://tailwindcss.com), [shadcn/ui](https://ui.shadcn.com) y [Radix](https://radix-ui.com), I7A/UI es un conjunto de componentes de interfaz de usuario que se pueden utilizar para crear aplicaciones web modernas y atractivas. Está diseñado para ser fácil de usar y personalizar, lo que permite a los desarrolladores centrarse en la creación de experiencias de usuario excepcionales.

## Motivación

Este proyecto nace de la curiosidad y la necesidad de construir una colección de componentes utiles tanto para mí como para otros desarrolladores que prefieren no partir desde cero al crear componentes. Su objetivo es ofrecer una base sólida para el desarrollo de aplicaciones web modernas, sin sacrificar flexibilidad ni personalización. Combinando buenas prácticas de diseño y desarrollo, I7A/UI busca simplificar la creación de interfaces de usuario atractivas, funcionales y adaptables.

## Código Abierto

I7A/UI es un proyecto de código abierto, lo que significa que cualquiera puede contribuir a su desarrollo y mejora. Si tienes ideas, sugerencias o mejoras, no dudes en abrir un issue o un pull request en el [repositorio](https://github.com/figueroaignacio/i7a-ui) de GitHub. Tu participación es muy apreciada y ayuda a construir una comunidad más fuerte en torno a I7A/UI.

## Preguntas Frecuentes

### ¿I7A/UI es una libreria de copiar y pegar?

Bueno, no exactamente. Este proyecto busca proporcionar componentes reutilizables y personalizables para copiar y pegar, pero no es una "libreria" ya que técnicamente no actua como dependencia.

### ¿Los componentes son unicamente para proyectos construidos en React?

No. Aunque los componentes están construidos en React, puedes adaptarlos a cualquier framework o libreria de tu preferencia. La idea es que puedas reutilizar el código y adaptarlo a tus necesidades.
